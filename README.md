3. 깃 업로드

- 깃허브 가입 & 로그인

```
git add .
```

```
git commit -m "first commit **[커밋 메시지]"
```

```
git branch -M main
```

```
git remote add origin https://github.com/h662/MERN.git **[깃허브주소]
```

```
git push -u origin main
```

이후에는 작업할 때마다 add, commit, push를 해주면 된다.
